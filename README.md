AMTweaker
=========

A Minecraft mod to support [MineTweaker](http://minetweaker3.powerofbytes.com/) customization of defeatedcrow's [Apple, Milk & Tea 2](http://forum.minecraftuser.jp/viewtopic.php?t=17657)
